# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to uncover purchasing patterns, customer preferences, and sales trends. Using Excel, SQL, Python, and Power BI, the project transforms raw transactional data into actionable business insights through data cleaning, analysis, and interactive dashboards.

## Objectives
- Analyze customer purchasing behavior.
- Identify top-performing product categories.
- Understand customer demographics and preferences.
- Discover sales trends and business opportunities.
- Create an interactive dashboard for decision-making.

## Tools & Technologies
- Microsoft Excel
- SQL
- Python
  - Pandas
  - NumPy
- Power BI

## Dataset
The dataset contains customer shopping information including:
- Customer demographics
- Product categories
- Purchase amounts
- Payment methods
- Shopping frequency
- Customer ratings and preferences

## Project Workflow

### 1. Data Collection & Cleaning
- Removed duplicate records.
- Handled missing values.
- Standardized data formats.
- Performed data validation checks.

### 2. SQL Analysis
Performed SQL queries to:
- Calculate total sales and revenue.
- Analyze category-wise performance.
- Identify customer purchasing patterns.
- Generate business metrics.

### 3. Python Analysis
Conducted Exploratory Data Analysis (EDA):
- Data preprocessing
- Statistical analysis
- Trend identification
- Data visualization

### 4. Power BI Dashboard
Developed an interactive dashboard featuring:
- Sales Overview
- Customer Demographics
- Product Category Performance
- Payment Method Analysis
- Customer Behavior Insights

## Key Insights
- Identified the highest revenue-generating product categories.
- Analyzed customer spending patterns.
- Determined preferred payment methods.
- Evaluated customer demographics and shopping habits.
- Generated actionable recommendations for business growth.
